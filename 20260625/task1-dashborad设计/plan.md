# Task 1: Dashboard 设计与主页开发

## 本 task 要完成的内容

以项目门户为核心，基于参考样式重构并搭建 KnowledgeMap 的主页：

1. **界面风格参考**：参考 `frontend_example` 文件夹下的静态 HTML/CSS 模版设计，实现具有高质感、现代暗色调微发光的产品门户。
2. **动画效果**：
   - 页面级平滑滚动（集成 Lenis 滚动库）。
   - 节点淡入动效：当页面向下滚动时，利用 Intersection Observer 实现元素的淡入与位移呈现（`useScrollReveal` 动效）。
   - 卡片微光过渡：项目卡片及 Journal 卡片悬停时有轻微浮起、发光和线条扫光动效。
3. **视觉配色**：
   - 默认采用深色模式（Dark Mode），背景色为深邃灰 `#0f0f14`，卡片采用碳黑 `#16161e`，搭配亮紫色与青色作为点缀微光。
   - 亮色模式采用温润米白 `#fcfbfa`（纸张质感），搭配 `#f4f2ee` 卡片背景与暗色边框。
4. **布局结构**：
   - **Hero Panel**：简洁的标题、滚动状态卡片以及进入主力模块的 CTA 按钮。
   - **Project Grid**：可交互的项目网格卡片，高亮标记可用入口（如记账与 KnowledgeMap），预留未接入模块的灰色虚位。
   - **Journal Section**：三栏式卡片，展示分类标签、标题、摘要及作者信息。
   - **About Section**：两栏式作者卡片展示区。
5. **图片和 GIF 引用**：
   - 将 `/images/pic1.png`、`/images/pic2.png`、`/images/pic3.png` 挂载在 Journal Section 的三个三栏卡片大图区。
   - 将 `/images/pic4.gif` 作为动态头像，放置在文章末尾作者“张斯涵”旁（仅限小字部分的元信息）。
   - 将 `/images/pic5.jpg` 作为大图挂载在 About Section 的图片占位中，使用 CSS 的 `background-size: cover` 确保宽高比失衡时自适应裁剪。
   - 将 “《星光咖啡馆与死神之蝶》” 的栏目标题字号缩小一号，以避免在一屏中排版局促。
6. **主题切换**：
   - 提取所有色彩配置为全局 CSS 变量。
   - 顶部导航栏增加模式切换按钮，支持 Dark / Light 双外观一键流畅切换，并在刷新后读取 LocalStorage 持久化状态。

## 预期结果

1. 打开主页后，首屏展示精美的动画渐显和流畅滚动效果。
2. 记账（Bills）卡片和 KnowledgeMap（Notes）卡片显示为“可进入”，点击可路由跳转；其余卡片为置灰的“预留”状态。
3. Journal 版块三栏完美对称呈现，“《星光咖啡馆与死神之蝶》”字号恰当，pic4.gif 头像显示正常。
4. About 区域中 pic5.jpg 正常渲染并保持宽高比缩放裁剪。
5. 模式一键切换时，整页包括文字、边框、背景及发光阴影同步响应。

## 边界

1. 主页为静态展示，主要负责分流与模块跳转，不直接请求后端 API 数据。
2. Newsletter 订阅输入框仅实现前端样式，未接入真实邮箱订阅数据库。
