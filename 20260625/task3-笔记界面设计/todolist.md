## Phase 2: Verlet 物理引擎与核心力场动力学 (高代码量 / 极重要)

- [x] 2.1 建立 Verlet 物理循环 (updatePhysics)
      → 使用 `requestAnimationFrame` 递归驱动物理计算。
      → 引入 `damping` 机制维持动量消耗与摩擦力平衡。
- [x] 2.2 实现节点磁力斥力 (Repulsion) 与向心重力 (Gravity)
      → 计算两两节点之间的间距平方级排斥力。
      → 向心重力拉拢外围节点，保证整网始终稳定位于屏幕中央。
- [x] 2.3 重构拉伸弹簧引力 (Stretchy Links)
      → 中枢与中枢连线静止长度设为 `160px`，刚度较高，分离三个星团。
      → 中枢与叶子节点连线静止长度设为 `95px`，降低刚度系数（`0.025`），允许被外部飘移力柔软拉伸。
- [x] 2.4 实现节点硬碰撞箱检测与坐标推开 (Collision Resolver)
      → 设定中枢半径 `22px`，叶子节点半径 `15px`，Margin 缓冲区 `12px`。
      → 间距小于理论下限时，强制修正物理位置防止重叠，并折算速度阻尼衰减。
- [x] 2.5 建立理想外向夹角扇区对齐 (Sector Alignment)
      → 中枢 1 (左下方) 绑定理想角 `150°`；中枢 2 (右下方) 绑定 `30°`；中枢 3 (正上方) 绑定 `-90°`。
      → 在物理引擎中对叶子节点加入外向约束引力，强制节点只在外侧扇区摆动，杜绝连线穿过中枢骨架线。
- [x] 2.6 实现多频 Lissajous 合成飘移波
      → 取消原单频轨道圆周运动，换用多频正弦/余弦合成的随机摇摆运动。
      → 飘移速度调整为极其柔和的 `0.12`。
- [x] 2.7 实现触碰悬停/拖拽完全静止与防抖透明热区
      → 检测 hovered / dragged 状态，直接将其物理速度归零，并直接跳过坐标迭代更新。
      → 在 SVG node 渲染的首层添加 `r=32` 的完全透明点击捕捉圆形，提高鼠标抓取稳定性。
